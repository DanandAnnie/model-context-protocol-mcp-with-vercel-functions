Property Data MCP Server

A PropertyRadar-style property intelligence platform exposed as MCP tools. 27 tools for property lookup, owner research, mortgage analysis, foreclosure tracking, list building, and investment analysis.

MCP Tools (27 total)

Property Lookup & Search

property_lookup_by_address Free data PR API
Full property details by street address. Combines PropertyRadar, Zillow, ATTOM, flood zone, and school data.
property_lookup_by_apn PR API
Look up by Assessor Parcel Number with county assessor links.
property_search PR API
Search with filters: location, type, beds/baths, sqft, value, owner status, etc.
property_comps PR API
Comparable sales near a property for valuation.

Owner Information

owner_lookup PR API
Owner name, mailing address, ownership type, vesting.
owner_search PR API
Find all properties by owner name.
find_absentee_owners PR API
Properties where owner doesn't live at property.
find_corporate_owned PR API
Properties owned by corporations or LLCs.
find_trust_owned PR API
Properties held in trusts.
find_out_of_state_owners PR API
Properties with owners in a different state.

Mortgage & Loan Data

mortgage_details PR API
Lender, amount, rate, type, term, estimated balance & payment.
find_high_equity_properties PR API
Properties with high equity / low LTV.
find_free_and_clear PR API
Properties with no mortgage (100% equity).
find_arm_loans PR API
Properties with adjustable rate mortgages.

Foreclosure & Distress

foreclosure_details PR API
NOD/NTS details, auction date, default amount, trustee info.
find_preforeclosures PR API
Properties with Notice of Default filed.
find_auction_properties PR API
Properties scheduled for trustee sale/auction.
find_bank_owned PR API
REO / bank-owned properties.
find_tax_delinquent PR API
Properties with unpaid property taxes.

Sales & Transfers

transfer_history PR API
Full sale/transfer history with dates, prices, parties.
find_recent_sales PR API
Recently sold properties, cash buyer filter.

List Building

build_property_list PR API
Create targeted lists with 40+ filter criteria (like PropertyRadar's 285+ criteria list builder).
get_property_lists PR API
Retrieve all saved property lists.

Census data: population, income, home values, rents, vacancy rates.
flood_zone_check Free data
FEMA flood zone designation for any address.
nearby_schools Free data
Schools near a property from OpenStreetMap.
geocode_address Free data
Address-to-coordinates and reverse geocoding.
county_assessor_lookup Free data
County assessor website lookup URLs by APN.

Valuation & Investment

property_valuation Free data PR API
Multi-source valuation: Zillow, ATTOM, PropertyRadar, assessor data.
investment_analysis Free data
Cash flow, cap rate, cash-on-cash return, DSCR analysis.

Alerts & Monitoring

create_property_alert PR API
Set up monitoring for new listings, price changes, foreclosures, sales.
check_alerts PR API
Manually check for properties matching alert criteria.

Setup & Environment Variables

Configure these in your Vercel project settings or .env file:

PROPERTY_RADAR_API_TOKENRequired for PR API tools. Get from PropertyRadar Settings
RAPIDAPI_KEYOptional. Enables Zillow Zestimate data via RapidAPI
ATTOM_API_KEYOptional. Enables ATTOM property data
CENSUS_API_KEYOptional. Census Bureau API (works without key at lower rate limits)
HUD_API_KEYOptional. HUD Fair Market Rent data

Connect via MCP

Add to your MCP client config (Claude Desktop, Cursor, etc.):

{
  "mcpServers": {
    "property-data": {
      "url": "https://YOUR-DEPLOYMENT.vercel.app/mcp"
    }
  }
}
